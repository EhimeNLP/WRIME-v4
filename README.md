# WRIME-v4: アスペクトベースの日本語感情分析

日本語感情分析コーパス **WRIME** を拡張し、  
各文に含まれる単語に対して **感情極性（5段階）** と **感情強度（8感情×4段階）** のラベルを付与したデータセットを構築しました。

## データ概要

- **元データ**: WRIME (Kajiwara et al. 2021)
- **拡張内容**:
  - 単語レベルの感情極性ラベル付け（5段階）
  - 単語レベルの感情強度ラベル付け（喜び、悲しみ、期待、驚き、怒り、恐怖、嫌悪、信頼 の8感情、4段階）

## データフォーマット

データは JSON 形式で保存されています。各文は以下の形式です：

```jsonc
{
  "sentence_id": 1,
  "sentence": "ぼけっとしてたらこんな時間。チャリあるから食べにでたいのに…",
  "opinions": [
    {
      "selectedWords": "時間",
      "emotions": {
        "polarity": -1,   // 感情極性（-2=強いネガ, -1=弱いネガ, 0=ニュートラル, 1=弱いポジ, 2=強いポジ）
        "joy": 0,         // 0=なし, 1=弱, 2=中, 3=強
        "sadness": 1,
        "anticipation": 0,
        "surprise": 1,
        "anger": 0,
        "fear": 0,
        "disgust": 0,
        "trust": 0
      },
      "from": 11,        // 文中の開始位置（文字単位）
      "to": 13          // 文中の終了位置（文字単位）
    }
  ]
}
```

### 文レベルの情報

| パラメータ名 | 型 | 説明 |
|-------------|----|------|
| `sentence_id` | int | 文を一意に識別するID |
| `sentence` | string | アノテーション対象の日本語文 |
| `opinions` | list | 文中の注目単語と感情ラベルを格納したリスト |

### 意見（opinions）レベルの情報

| パラメータ名 | 型 | 説明 |
|-------------|----|------|
| `selectedWords` | string | 感情ラベルが付与された対象単語 |
| `emotions` | object | 感情ラベルを格納した辞書 |
| `emotions.polarity` | int | 感情極性（-2: 強ネガ, -1: ネガ, 0: 中立, 1: ポジ, 2: 強ポジ） |
| `emotions.joy` | int | 喜びの強度（0: なし, 1: 弱, 2: 中, 3: 強） |
| `emotions.sadness` | int | 悲しみの強度（0: なし, 1: 弱, 2: 中, 3: 強） |
| `emotions.anticipation` | int | 期待の強度（0: なし, 1: 弱, 2: 中, 3: 強） |
| `emotions.surprise` | int | 驚きの強度（0: なし, 1: 弱, 2: 中, 3: 強） |
| `emotions.anger` | int | 怒りの強度（0: なし, 1: 弱, 2: 中, 3: 強） |
| `emotions.fear` | int | 恐れの強度（0: なし, 1: 弱, 2: 中, 3: 強） |
| `emotions.disgust` | int | 嫌悪の強度（0: なし, 1: 弱, 2: 中, 3: 強） |
| `emotions.trust` | int | 信頼の強度（0: なし, 1: 弱, 2: 中, 3: 強） |
| `from` | int | 感情ラベルが付与された単語の開始文字位置（0始まり） |
| `to` | int | 感情ラベルが付与された単語の終了文字位置 |
